Descrição do Projeto

O projeto se trata de um sistema médico, com o intuito do usuário registrar
seu pedido de atendimento de forma remota, ou seja, informar suas necessidades
no momento do ocorrido e já registrar seu pedido, assim, podendo ser atendido
de forma correta quando chegar no consultório médico.

Sendo um "agendamento rápido" sem a necessidade de chegar no hospital para realiza-lo.

-> Esse projeto foi feito a pedido da faculdade em alguns semestres passados.


Informações Importantes

No banco de dados existem duas tabelas, elas são: 
"RegisterOrder" utilizada para registrar o pedido do usuário.
"RegisterUser" utilizada para registrar o usuário e suas informações.

Você pode criar uma conta nova para acessa-la ou pode entrar em duas contas que eu criei no SQL
As duas contas são:

Project Description

The project is a medical system, with the intention that the user registers
his request for care remotely, that is, inform your needs and already register your request, so you can be served
the correct way when he gets to the doctor's office.

It is a "quick consultation" without the need to arrive at the hospital to do it.

-> This project was carried out at the request of the faculty in some past semesters.


Important Information

In the database there are two tables, they are: 
"RegisterOrder" used to register the user's order.
"RegisterUser" used to register the user and their information.
